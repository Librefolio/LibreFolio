# F-047 вЂ” Transaction List Page

## What it does
The `/transactions` page showing all transactions the current user has access to across all their brokers. Built on the generic `DataTable` component with column header filters (multi-enum, currency-stack range sliders, date pickers, text search). Supports bulk delete, TransferPromote, and Staging Modal launch. TRANSFER/FX_CONVERSION pairs always rendered adjacent (giver above / receiver below) via the always-pair-adjacent algorithm (see [[concepts/always-pair-adjacent]]).

**Frontend**: `frontend/src/routes/(app)/transactions/+page.svelte`, `frontend/src/lib/components/transactions/TransactionsTable.svelte`, `frontend/src/lib/components/table/DataTable.svelte`

## Architecture notes

### Client-side filtering
All filtering is 100% client-side. `GET /transactions` loads ALL user transactions (no filter params). Explicit **Refresh** button (в†є) triggers reload. See [[decisions/transactions-client-side-filtering]].

### Always-pair-adjacent rendering
TRANSFER/FX_CONVERSION pairs always adjacent: giver above, receiver below. Secondary fetch for ghost rows when partner absent from main result. Conditional: only when no column filter and no sort active (with filter/sort в†’ `isGrouped=false`, independent rows). See [[concepts/always-pair-adjacent]].

### Entity store (asset + broker)
`assetStore` and `brokerStore` use `createEntityStore<T>()` factory for shared cache with proper invalidation. See [[concepts/entity-store-pattern]].

### Modals
- `TransactionFormModal.svelte` вЂ” single-row CRUD (create/edit/duplicate/view) + **dual-form mode** for paired transactions (FX, Transfer Asset, Transfer Cash). See [[decisions/dual-transaction-form-design]]
- `TransactionBulkModal.svelte` вЂ” DataTable-based bulk editor with editable cells, mixed row states (new/edit/delete). See [[features/F-048]]
- `PromotePairWizardModal.svelte` вЂ” 3-step wizard for pair promotion (replaces legacy `TransferPromoteModal`)
- `BulkDeleteLinkedPairModal.svelte` вЂ” bulk delete with pair-aware confirmation

### New filter variants (DataTable)
- `multi-enum` вЂ” checkbox list with search (transaction type, tags)
- `currency-stack` вЂ” per-currency range sliders with flag+symbol+code title (`getCurrencyMinMaxByCode(column)`)
- `DataTable.fullData` prop вЂ” stable min/max boundaries (don't shrink as filter narrows)

### Tooltip rule
All tooltips inside DataTable cells use `<Tooltip.svelte>` via CustomCell. `title=""` HTML attributes prohibited. See [[decisions/datatable-tooltip-custom-cell]].

### Pulse animation (GoTo linked pair)
Click `рџ”—` icon в†’ DOM-direct `classList.add/remove` + `offsetWidth` reflow to restart CSS animation on `<td>` (`tr.tx-row-highlight > td`). Capture-phase event delegation for `рџ”—`/`в—Џevt` click events.
