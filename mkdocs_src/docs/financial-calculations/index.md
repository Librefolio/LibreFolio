# Financial Calculations

This section documents the mathematical formulas and conventions used in LibreFolio for financial calculations.

## Overview

Accurate financial calculations are critical for a portfolio tracker. LibreFolio implements standard financial conventions to ensuring consistency with broker reports and real-world data.

## Key Concepts

-   **[Day Count Conventions](day-count.md)**: How the number of days between two dates is calculated (e.g., for interest accrual).
-   **[Interest Types](interest-types.md)**: The difference between Simple and Compound interest.
-   **[Compounding](compounding.md)**: How compounding frequency affects the final value.
