# <img src="../../../static/icons/asset-types/other.png" width="32" style="vertical-align: middle;" /> Otros

El tipo de activo **Otros** es una categoría general para cualquier clase de activo que no esté cubierta por los tipos estándar. Proporciona la máxima flexibilidad para el seguimiento de inversiones no tradicionales.

---

## 🔑 Características Clave

| Propiedad | Detalle |
|----------|--------|
| **Código en LibreFolio** | `OTHER` |
| **Valoración** | Ingreso manual o cualquier proveedor compatible |
| **Moneda** | Cualquier moneda ISO 4217 |
| **Ingresos** | Depende del activo específico |
| **Proveedores típicos** | Manual, CSS Scraper |

---

## 📊 Ejemplos de Casos de Uso

| Activo | Descripción |
|-------|-------------|
| **Arte y Coleccionables** | Pinturas, vino, monedas raras — valorados mediante tasaciones |
| **Capital Privado (Private Equity)** | Acciones en empresas no públicas |
| **Capital Riesgo (Venture Capital)** | Inversiones en startups en etapas tempranas |
| **Relojes y Joyas** | Artículos de lujo mantenidos como inversiones |
| **Nombres de Dominio** | Activos digitales con valor de reventa |
| **Créditos de Carbono** | Certificados de compensación ambiental |

---

## 🔧 Seguimiento en LibreFolio

Para activos sin un flujo de precios público:

1. Use el proveedor **Manual** e ingrese los precios a través del [Editor de Datos](../../../user/assets/detail/data-editor.md)
2. O use el proveedor **CSS Scraper** para extraer precios de un sitio web
3. Registre las transacciones de compra/venta como de costumbre a través de las importaciones del bróker
