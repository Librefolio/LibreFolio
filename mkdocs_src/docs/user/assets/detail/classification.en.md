# 🗂️ Classification

The Classification panel shows the asset's **sector** and **geographic** distribution, when this data has been configured for the asset (manually or fetched from a provider).

## 📊 Sector Distribution

A **pie chart** showing the breakdown by industry sector (e.g., Technology 35%, Healthcare 20%, Finance 15%...). This is especially useful for ETFs and diversified funds to understand the underlying sector allocation.

## 🌍 Geographic Distribution

### World Map

An **interactive world map** with countries colored by allocation weight. Hover over a country to see its percentage.

### Country Breakdown

A **pie chart** showing the percentage breakdown by country or region (e.g., United States 60%, Europe 25%, Asia 15%...).

---

## 🛠️ How to Use

1. Click the **Classification** toggle button in the toolbar area
2. The panel expands below the chart showing sector and geographic data
3. Hover over pie chart slices or map regions for details

---

## ⚙️ Configuring Classification

Classification data can be set in two ways:

1. **Provider metadata**: Some providers (Yahoo Finance, justETF) fetch sector and geographic data automatically during sync
2. **Manual entry**: Edit the asset (✏️) and fill in the sector/geographic distributions using the **Distribution Editor**
