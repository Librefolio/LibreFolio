# 🗂️ Classification

Le panneau de Classification affiche la répartition **sectorielle** et **géographique** de l'actif, lorsque ces données ont été configurées pour l'actif (manuellement ou récupérées via un fournisseur).

## 📊 Répartition Sectorielle

Un **graphique circulaire** montrant la ventilation par secteur d'activité (ex: Technologie 35%, Santé 20%, Finance 15%...). C'est particulièrement utile pour les ETF et les fonds diversifiés afin de comprendre l'allocation sectorielle sous-jacente.

## 🌍 Répartition Géographique

### Carte du Monde

Une **carte du monde interactive** où les pays sont colorés selon le poids de l'allocation. Survolez un pays pour voir son pourcentage.

### Ventilation par Pays

Un **graphique circulaire** montrant la ventilation en pourcentage par pays ou région (ex: États-Unis 60%, Europe 25%, Asie 15%...).

---

## 🛠️ Comment l'utiliser

1. Cliquez sur l'interrupteur **Classification** dans la barre d'outils
2. Le panneau se développe sous le graphique pour afficher les données sectorielles et géographiques
3. Survolez les segments du graphique circulaire ou les régions de la carte pour plus de détails

---

## ⚙️ Configurer la Classification

Les données de classification peuvent être définies de deux manières :

1. **Métadonnées du fournisseur** : Certains fournisseurs (Yahoo Finance, justETF) récupèrent automatiquement les données sectorielles et géographiques lors de la synchronisation
2. **Saisie manuelle** : Modifiez l'actif (✏️) et remplissez les répartitions sectorielles/géographiques à l'aide de l'**Éditeur de distribution**
