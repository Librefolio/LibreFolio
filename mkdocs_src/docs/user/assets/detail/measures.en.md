# 📐 Measures

The Measure tool lets you interactively measure price differences, percentage changes, and time intervals between any two points on the chart.

## 🛠️ How to Use

1. Click the **Measure** button (📐) in the toolbar
2. The measure panel opens below the chart
3. Click a **start point** on the chart
4. Click an **end point** on the chart
5. The panel displays:
    - 📊 **Price delta** — absolute price difference
    - 📈 **Percentage change** — relative change between the two points
    - 📅 **Time interval** — number of days between the two points
    - 📉 **Annualized return** — the return projected to a yearly basis
