# 📐 Mesures

L'outil de mesure vous permet de mesurer interactivement les différences de prix, les variations en pourcentage et les intervalles de temps entre deux points quelconques sur le graphique.

## 🛠️ Mode d'emploi

1. Cliquez sur le bouton **Mesure** (📐) dans la barre d'outils
2. Le panneau de mesure s'ouvre sous le graphique
3. Cliquez sur un **point de départ** sur le graphique
4. Cliquez sur un **point d'arrivée** sur le graphique
5. Le panneau affiche :
 - 📊 **Variation de prix** — différence de prix absolue
 - 📈 **Variation en pourcentage** — variation relative entre les deux points
 - 📅 **Intervalle de temps** — nombre de jours entre les deux points
 - 📉 **Rendement annualisé** — le rendement projeté sur une base annuelle
