# 💼 Actifs

Les actifs sont le cœur de LibreFolio. Ils représentent tout instrument financier que vous possédez ou suivez : actions, ETF, obligations, cryptomonnaies ou instruments personnalisés comme des comptes d'épargne avec intérêts programmés.

## 📌 Qu'est-ce qu'un actif ?

Un actif dans LibreFolio est un instrument financier comprenant :

- **Identité** : nom, ISIN, ticker ou autres identifiants
- **Catégorie** : action, ETF, obligation, crypto, matière première, etc.
- **Devise** : la devise dans laquelle l'actif est libellé
- **Fournisseur** : un fournisseur de prix optionnel qui récupère automatiquement les cours actuels et l'historique
- **Classification** : distribution sectorielle et géographique (graphiques en secteurs + carte du monde)
- **Transactions** : opérations d'achat, de vente, de perception de dividendes et d'intérêts liées à un portefeuille

## 📋 Liste des actifs

Accédez à **Actifs** dans la barre latérale pour voir tous vos actifs. La page de liste propose :

- **Recherche** : filtrer les actifs par nom, ISIN ou ticker
- **Filtre de type** : afficher uniquement des types d'actifs spécifiques (ETF, Action, Obligation, etc.)
- **Interrupteur Actifs/Tous** : afficher uniquement les actifs actifs ou inclure ceux archivés
- **Tout synchroniser** : récupérer les derniers prix pour tous les actifs disposant d'un fournisseur
- **Tout actualiser** : recharger les données des actifs depuis la base de données

Cliquez sur n'importe quelle carte d'actif pour accéder à sa **[page de détail](detail/index.md)**.
