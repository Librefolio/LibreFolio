# 📊 justETF Provider

justETF provides detailed data for European ETFs, including current prices and historical data.

## 📊 Capabilities

- ✅ **Current Price**: Daily NAV and market price
- ✅ **History**: Historical price data
- ❌ **Search**: Not supported (use ISIN directly)

## 🔧 Configuration

- **Identifier**: ISIN code (e.g., `IE00BK5BQT80`)
- **Identifier Type**: `ISIN`
- **Parameters**: None required

## 💡 Examples

| Asset | ISIN |
|-------|------|
| Vanguard FTSE All-World | `IE00BK5BQT80` |
| iShares Core MSCI World | `IE00B4L5Y983` |
| Xtrackers MSCI Emerging Markets | `IE00BTJRMP35` |

## 📝 Notes

- Best suited for European-domiciled ETFs listed on justETF
- Uses the ISIN as the primary identifier
