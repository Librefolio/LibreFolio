# 📊 Proveedor justETF

justETF proporciona datos detallados para ETF europeos, incluyendo precios actuales y datos históricos.

## 📊 Capacidades

- ✅ **Precio Actual**: NAV diario y precio de mercado
- ✅ **Historial**: Datos de precios históricos
- ❌ **Búsqueda**: No disponible (use el ISIN directamente)

## 🔧 Configuración

- **Identificador**: Código ISIN (ej. `IE00BK5BQT80`)
- **Tipo de Identificador**: `ISIN`
- **Parámetros**: No se requieren

## 💡 Ejemplos

| Activo | ISIN |
|-------|------|
| Vanguard FTSE All-World | `IE00BK5BQT80` |
| iShares Core MSCI World | `IE00B4L5Y983` |
| Xtrackers MSCI Emerging Markets | `IE00BTJRMP35` |

## 📝 Notas

- Ideal para ETF domiciliados en Europa listados en justETF
- Utiliza el ISIN como identificador principal
