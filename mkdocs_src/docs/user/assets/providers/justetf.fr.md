# 📊 Fournisseur justETF

justETF fournit des données détaillées pour les ETF européens, incluant les prix actuels et les données historiques.

## 📊 Fonctionnalités

- ✅ **Prix actuel** : Valeur Liquidative (NAV) quotidienne et prix du marché
- ✅ **Historique** : Données de prix historiques
- ❌ **Recherche** : Non prise en charge (utilisez directement l'ISIN)

## 🔧 Configuration

- **Identifiant** : Code ISIN (ex: `IE00BK5BQT80`)
- **Type d'identifiant** : `ISIN`
- **Paramètres** : Aucun

## 💡 Exemples

| Actif | ISIN |
|-------|------|
| Vanguard FTSE All-World | `IE00BK5BQT80` |
| iShares Core MSCI World | `IE00B4L5Y983` |
| Xtrackers MSCI Emerging Markets | `IE00BTJRMP35` |

## 📝 Notes

- Idéal pour les ETF domiciliés en Europe et répertoriés sur justETF
- Utilisation de l'ISIN comme identifiant principal
