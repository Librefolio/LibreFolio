# 🏦 Broker

Un **Broker** in LibreFolio rappresenta un conto di intermediazione — dove risiedono i tuoi investimenti (ad es. Interactive Brokers, Degiro, un conto bancario).

Tutte le transazioni, i report e i dati di importazione sono collegati a un broker. È necessario almeno un broker per iniziare a monitorare il proprio portafoglio.

## ➕ Creazione di un Broker

1. Vai alla pagina **Broker** dalla barra laterale
2. Clicca su **"New Broker"**
3. Compila i dettagli: nome, valuta di base e, opzionalmente, un'icona

## 🔧 Cosa puoi fare con un Broker?

| Funzionalità | Descrizione | Stato |
|---------|-------------|--------|
| **Visualizza dettagli** | Nome, valuta, data di creazione, icona | ✅ Disponibile |
| **Carica report** | File CSV/Excel dal tuo broker reale | ✅ Disponibile |
| **Condividi accesso** | Invita altri utenti con permessi basati sui ruoli | ✅ Disponibile |
| **Importa transazioni** | Parsing automatico dei report caricati tramite BRIM | 🔜 Prossimamente |
| **Visualizza transazioni** | Elenco, filtro e gestione delle transazioni importate | 🔜 Prossimamente |

!!! note "Work in Progress"

    La pagina dei dettagli del broker e l'interfaccia di gestione delle transazioni sono in fase di sviluppo attivo. Attualmente è possibile creare broker, caricare report e condividere l'accesso. L'importazione completa e la visualizzazione delle transazioni verranno aggiunte nelle prossime fasi.
