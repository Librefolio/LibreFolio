## Aperçu

Le Tableau de bord est la **page d'accueil principale** après la connexion. Il fournit une vue consolidée de l'ensemble de votre portefeuille pour tous les courtiers — valeur nette, allocation, performances récentes et widgets d'accès rapide.

Les widgets prévus incluent :

- **Valeur nette** — Valeur totale du portefeuille dans votre devise de référence
- **Allocation** — Graphiques de distribution sectorielle et par type d'actif
- **Aperçu des Performances** — Rendements sur des plages temporelles sélectionnables (1S, 1M, 3M, YTD, 1A, Total)
- **Transactions Récentes** — Les 5 à 10 dernières transactions pour tous les courtiers
- **Mouvements du Marché** — Vos positions triées par variation quotidienne en %
- **Exposition FX** — Résumé de l'exposition aux devises

---

## Vérification de la Connectivité

LibreFolio vérifie automatiquement la **connectivité du backend** lorsque vous ouvrez le Tableau de bord. Si le backend est inaccessible, une bannière s'affiche avec un bouton de reconnexion — aucune donnée obsolète n'est affichée.
