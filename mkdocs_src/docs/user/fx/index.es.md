# 💱 FX Rates (Tipo de Cambio)

LibreFolio incluye un sistema de gestión de **Foreign Exchange (FX)** completo. Permite realizar el seguimiento de los tipos de cambio entre cualquier par de divisas, con sincronización automática desde fuentes oficiales de bancos centrales.

---

## 📋 Lista de FX

Navegue a **Tipos de cambio** desde la barra lateral para ver todos sus pares de divisas configurados:

<div class="screenshot-container" style="max-width: 700px; margin: 1rem auto;">
 <img class="gallery-img" data-category="fx" data-name="list" alt="Página de Lista de FX" style="width: 100%; border-radius: 8px; box-shadow: 0 4px 16px rgba(0,0,0,0.1);">
</div>

Cada tarjeta muestra:

- 🏷️ El **par de divisas** con emojis de banderas (ej., 🇪🇺 EUR → 🇺🇸 USD)
- 📈 El **tipo de cambio más reciente** y la tendencia
- 🏛️ El **proveedor de datos** (ECB, FED, BOE, etc.)
- 📊 Un **mini gráfico sparkline**

Puede **filtrar** por divisa para encontrar rápidamente un par específico:
