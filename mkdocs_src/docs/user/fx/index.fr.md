# 💱 Taux FX (Change de Devises)

LibreFolio inclut un système complet de gestion du **change (FX)**. Il vous permet de suivre les taux de change entre n'importe quelle paire de devises, avec une synchronisation automatique à partir de sources officielles de banques centrales.

---

## 📋 La page de liste des taux FX

Naviguez vers **Taux FX** depuis la barre latérale pour voir toutes vos paires de devises configurées :

<div class="screenshot-container" style="max-width: 700px; margin: 1rem auto;">
 <img class="gallery-img" data-category="fx" data-name="list" alt="Page de liste FX" style="width: 100%; border-radius: 8px; box-shadow: 0 4px 16px rgba(0,0,0,0.1);">
</div>

Chaque carte affiche :

- 🏷️ La **paire de devises** avec des emojis de drapeaux (ex: 🇪🇺 EUR → 🇺🇸 USD)
- 📈 Le **dernier taux** et la tendance
- 🏛️ Le **fournisseur de données** (ECB, FED, BOE, etc.)
- 📊 Un **mini graphique** (sparkline)

Vous pouvez **filtrer** par devise pour trouver rapidement une paire spécifique :
