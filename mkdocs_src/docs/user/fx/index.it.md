# 💱 Tassi FX (Cambio Valute)

LibreFolio include un sistema completo di gestione del **Foreign Exchange (FX)**. Consente di monitorare i tassi di cambio tra qualsiasi coppia di valute, con sincronizzazione automatica da fonti ufficiali delle banche centrali.

---

## 📋 La Pagina Elenco FX

Naviga su **FX Rates** dalla barra laterale per visualizzare tutte le coppie di valute configurate:

<div class="screenshot-container" style="max-width: 700px; margin: 1rem auto;">
 <img class="gallery-img" data-category="fx" data-name="list" alt="Pagina Elenco FX" style="width: 100%; border-radius: 8px; box-shadow: 0 4px 16px rgba(0,0,0,0.1);">
</div>

Ogni scheda mostra:

- 🏷️ La **coppia di valute** con le emoji delle bandiere (es. 🇪🇺 EUR → 🇺🇸 USD)
- 📈 Il **tasso più recente** e il trend
- 🏛️ Il **provider** (ECB, FED, BOE, ecc.)
- 📊 Un **mini grafico sparkline**

È possibile **filtrare** per valuta per trovare rapidamente una coppia specifica:
