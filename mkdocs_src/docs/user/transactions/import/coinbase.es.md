# 🏦 Coinbase

!!! info "Beta"

    Este plugin está en **Beta** — se ha probado con archivos de muestra, pero podrían existir casos aislados.

## Cómo exportar

1. Inicie sesión en [Coinbase](https://www.coinbase.com).
2. Vaya a **Taxes → Reports → Generate Report → Transaction History** y exporte como CSV.

## Notas

- Solo transacciones de criptomonedas.
- Se admiten USD y EUR como monedas base.
