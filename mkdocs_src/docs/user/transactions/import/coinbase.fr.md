# 🏦 Coinbase

!!! info "Bêta"

    Ce plugin est en **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Coinbase](https://www.coinbase.com).
2. Allez dans **Taxes → Reports → Generate Report → Transaction History** et exportez au format CSV.

## Notes

- Transactions crypto uniquement.
- Devises de base USD et EUR prises en charge.
