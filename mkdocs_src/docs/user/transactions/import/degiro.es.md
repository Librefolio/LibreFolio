# 🏦 Degiro

!!! info "Beta"

    Este plugin está en fase **Beta**: ha sido probado con archivos de muestra, pero podrían existir casos aislados no contemplados.

## Cómo exportar

1. Inicie sesión en [Degiro](https://www.degiro.eu).
2. Vaya a **Cuenta → Informes (Account → Reports)** y exporte su **Estado de cuenta (Account Statement)** como CSV.

## Notas

- Se admiten informes en varios idiomas (EN, NL, IT, etc.).
- Los dividendos y las transacciones se encuentran en secciones separadas; cargue ambos archivos si es necesario.
