# 🏦 Degiro

!!! info "Bêta"

    Ce plugin est en **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Degiro](https://www.degiro.eu).
2. Allez dans **Compte → Rapports** et exportez votre **Relevé de compte** au format CSV.

## Notes

- Les rapports multilingues sont pris en charge (EN, NL, IT, etc.).
- Les dividendes et les transactions se trouvent dans des sections distinctes — importez les deux si nécessaire.
