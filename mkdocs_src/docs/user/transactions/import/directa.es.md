# 🏦 Directa SIM

!!! info "Beta"

    Este plugin está en **Beta**: se ha probado con archivos de muestra, pero pueden existir casos aislados.

## Cómo exportar

1. Inicie sesión en [Directa SIM](https://www.directatrading.com).
2. Vaya a **Operazioni → Movimenti** y exporte como CSV.

## Notas

- Bróker italiano. La exportación tiene un encabezado de 9 líneas que se omite automáticamente.
- Soporta transacciones de acciones y ETF.
