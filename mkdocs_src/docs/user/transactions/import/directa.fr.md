# 🏦 Directa SIM

!!! info "Bêta"

    Ce plugin est en **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Directa SIM](https://www.directatrading.com).
2. Allez dans **Operazioni → Movimenti** et exportez au format CSV.

## Notes

- Courtier italien. L'exportation possède un en-tête de 9 lignes qui est automatiquement ignoré.
- Prend en charge les transactions d'actions et d'ETF.
