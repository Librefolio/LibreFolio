# 🏦 Directa SIM

!!! info "Beta"

    Questo plugin è in versione **Beta** — testato con file di esempio, ma potrebbero esserci dei casi limite.

## Come Esportare

1. Accedi a [Directa SIM](https://www.directatrading.com).
2. Vai su **Operazioni → Movimenti** ed esporta in CSV.

## Note

- Broker italiano. L'export presenta un'intestazione di 9 righe che viene saltata automaticamente.
- Supporta transazioni su azioni ed ETF.
