# 🏦 eToro

!!! info "Beta"

    Este plugin está en fase **Beta**: se ha probado con archivos de muestra, pero podrían existir casos excepcionales.

## Cómo exportar

1. Inicie sesión en [eToro](https://www.etoro.com).
2. Vaya a **Cartera → History → Account Statement** y exporte como CSV.

## Notas

- Soporta transacciones de acciones y CFD.
- Las transacciones con criptomonedas pueden tener detalles limitados.
