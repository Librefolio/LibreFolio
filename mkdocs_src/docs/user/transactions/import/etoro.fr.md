# 🏦 eToro

!!! info "Bêta"

    Ce plugin est en phase **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [eToro](https://www.etoro.com).
2. Allez dans **Portefeuille → History → Account Statement** et exportez au format CSV.

## Notes

- Prend en charge les transactions d'actions et de CFD.
- Les transactions crypto peuvent comporter des détails limités.
