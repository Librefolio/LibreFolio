# 🏦 Finpension

!!! info "Beta"

    Este plugin está en fase **Beta**: se ha probado con archivos de muestra, pero pueden existir casos límite.

## Cómo exportar

1. Inicie sesión en [Finpension](https://www.finpension.ch).
2. Vaya a **Account → Transactions → Export** y descargue el CSV.

## Notas

- Plataforma de pensiones suiza.
- Utiliza el punto y coma (`;`) como delimitador de CSV; el plugin gestiona esto automáticamente.
