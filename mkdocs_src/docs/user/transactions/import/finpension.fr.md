# 🏦 Finpension

!!! info "Bêta"

    Ce plugin est en version **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Finpension](https://www.finpension.ch).
2. Allez dans **Compte → Transactions → Export** et téléchargez le CSV.

## Notes

- Plateforme de prévoyance suisse.
- Utilise le point-virgule (`;`) comme délimiteur CSV — le plugin gère cela automatiquement.
