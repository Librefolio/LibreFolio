# 🏦 Freetrade

!!! info "Beta"

    This plugin is in **Beta** — tested with sample files but edge cases may exist.

## How to Export

1. Log in to [Freetrade](https://www.freetrade.io).
2. Go to **Account → Activity → Export** and download the CSV.

## Notes

- UK broker. GBP and USD instruments supported.
