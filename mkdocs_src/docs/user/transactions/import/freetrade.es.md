# 🏦 Freetrade

!!! info "Beta"

    Este plugin está en **Beta** — probado con archivos de muestra, pero pueden existir casos límite.

## Cómo Exportar

1. Inicie sesión en [Freetrade](https://www.freetrade.io).
2. Vaya a **Account → Activity → Export** y descargue el CSV.

## Notas

- Bróker del Reino Unido. Instrumentos en GBP y USD compatibles.
