# 🏦 Interactive Brokers (IBKR)

!!! info "Beta"

    Questo plugin è in **Beta** — testato con file di esempio, ma potrebbero esserci casi limite.

## Come Esportare

1. Accedi al [Portale Clienti di Interactive Brokers](https://www.interactivebrokers.com) o a Trader Workstation (TWS).
2. Vai su **Reports → Activity → Flex Queries** oppure **Statements → Activity Statement**.
3. Seleziona l'intervallo di date ed esporta in formato **CSV**.

## Note

- Supporta i report di attività standard di IBKR (transazioni, dividendi, commissioni, depositi, prelievi).
- I conti multi-valuta sono supportati.
- Le azioni societarie (frazionamenti, fusioni) potrebbero richiedere un aggiustamento manuale.
