# 🏦 Revolut

!!! info "Beta"

    Este plugin está en fase **Beta**: ha sido probado con archivos de muestra, pero podrían existir casos excepcionales.

## Cómo exportar

1. Inicie sesión en [Revolut](https://www.revolut.com).
2. Vaya a **Stocks → Statements** y exporte el historial de transacciones como CSV.

## Notas

- Solo para cuentas de Revolut Trading (no para la cuenta principal de la tarjeta).
- Los importes en múltiples divisas dentro del mismo archivo se gestionan automáticamente.
