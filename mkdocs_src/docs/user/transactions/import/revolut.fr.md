# 🏦 Revolut

!!! info "Bêta"

    Ce plugin est en **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Revolut](https://www.revolut.com).
2. Allez dans **Actions → Relevés** et exportez l'historique des transactions au format CSV.

## Notes

- Comptes Revolut Trading uniquement (pas le compte courant principal).
- Les montants multi-devises dans un même fichier sont gérés automatiquement.
