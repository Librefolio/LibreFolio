# 🏦 Charles Schwab

!!! info "Beta"

    Este plugin está en fase **Beta** — ha sido probado con archivos de muestra, pero podrían existir casos excepcionales.

## Cómo exportar

1. Inicie sesión en [Charles Schwab](https://www.schwab.com).
2. Vaya a **Accounts → History → Export** y descargue el archivo como CSV.

## Notas

- CSV con formato de EE. UU. (fechas MM/DD/YYYY, importes en USD).
- Soporta transacciones de renta variable y dividendos.
