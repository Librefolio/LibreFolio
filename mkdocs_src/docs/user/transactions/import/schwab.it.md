# 🏦 Charles Schwab

!!! info "Beta"

    Questo plugin è in versione **Beta** — testato con file di esempio, ma potrebbero esserci casi limite.

## Come esportare

1. Accedi a [Charles Schwab](https://www.schwab.com).
2. Vai su **Accounts → History → Export** e scarica il file in formato CSV.

## Note

- CSV in formato statunitense (date MM/GG/AAAA, importi in USD).
- Supportate le transazioni di equity e i dividendi.
