# 🏦 Trading212

!!! info "Beta"

    Este plugin está en fase **Beta**: se ha probado con archivos de muestra, pero pueden existir casos excepcionales.

## Cómo exportar

1. Inicie sesión en [Trading212](https://www.trading212.com).
2. Vaya a **History → Export** y descargue el CSV.

## Notas

- Se admiten transacciones de acciones, ETF y PIE.
