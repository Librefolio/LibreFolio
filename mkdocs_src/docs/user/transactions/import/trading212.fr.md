# 🏦 Trading212

!!! info "Bêta"

    Ce plugin est en **Bêta** — testé avec des fichiers d'exemple, mais des cas particuliers peuvent exister.

## Comment exporter

1. Connectez-vous à [Trading212](https://www.trading212.com).
2. Allez dans **History → Export** et téléchargez le CSV.

## Notes

- Les transactions sur les actions, ETF et PIE sont prises en charge.
