# 🏦 Trading212

!!! info "Beta"

    Questo plugin è in versione **Beta** — testato con file di esempio, ma potrebbero esserci casi limite.

## Come Esportare

1. Accedi a [Trading212](https://www.trading212.com).
2. Vai su **History → Export** e scarica il CSV.

## Note

- Supportate le transazioni di azioni, ETF e PIE.
